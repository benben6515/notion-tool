## Tech stack & Dev

- React
- Tailwind
- Vite

Addition, We use [pnpm](https://pnpm.io/) for node packages manager.

```sh
# first init
git clone https://github.com/benben6515/notion-tool.git

# if you don't install pnpm yet
brew install pnpm
# or use npm
npm i -g pnpm

# install dependency
pnpm i

# dev
pnpm dev

# happing coding
```
