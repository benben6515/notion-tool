## 技術棧 & 開發

- React
- Tailwind
- Vite

對了，我們還使用 [pnpm](https://pnpm.io/) 做為 Node.js 套件管理

```sh
# 第一次初始化
git clone https://github.com/benben6515/notion-tool.git

# 如果還沒安裝過 pnpm
brew install pnpm
# 或是使用 npm 安裝 pnpm
npm i -g pnpm

# 安裝專案套件
pnpm i

# 開發
pnpm dev

# 快樂寫扣
```
